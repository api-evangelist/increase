# Increase (increase)

Increase is a banking and payments API providing direct ACH, wire, RTP, FedNow, check, and card programs. Bank-direct connectivity (no aggregator middle layer).

**APIs.json:** [https://raw.githubusercontent.com/api-evangelist/increase/refs/heads/main/apis.yml](https://raw.githubusercontent.com/api-evangelist/increase/refs/heads/main/apis.yml)
